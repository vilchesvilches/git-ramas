GIT-Ramas

En este proyecto se muestra el uso de ramas en GIT, para ello se han creado dos ramas: "main" y "feature".

La rama "main" es la rama principal del proyecto, mientras que la rama "feature" es una rama de desarrollo donde se han realizado cambios y mejoras.

Para crear una rama en GIT, se utiliza el comando git branch nombre-de-la-rama. Para cambiar a una rama, se utiliza el comando git checkout nombre-de-la-rama.

Una vez que se han realizado los cambios en la rama "feature", se pueden fusionar con la rama "main" utilizando el comando git merge feature.
